# YASC: Yet Another SDVX Controller

YASC is my attempt to make a cheap, open source, easy to build, yet effective Sound Voltex controller for use at home.

## BOM: 
2x EN16-H20AF15 Encoders (Highly suggest buying extras, my DigiKey order had quite a few faulty ones)\n
7x Mechanical keyboard switches of your choice
1x Pi Pico (use a fullsize one, I prefer using one with USB-C)
4x M3 brass threaded inserts
4x M3x16 SHCS screws
Enough wire to put everything together
